# Aufgabe

Erweitere die HTML Datei um eine Input Box, in die der Benutzer seinen Namen eintragen kann:

```html
<input type="text">
```

Wenn der Benutzer dann den bereits vorhandenen Button klickt, dann soll der Begrüßungtext auch seinen eingetragenen Namen beinhalten, z.B. "Hallo Max! Du hast gerade zum ersten mal JavaScript benutzt!".
Wenn das Textfeld leer ist, dann soll stattdessen "Gib erst deinen Namen ein." ausgegeben werden.

Den Wert der Input-Box erhält man, mit der Eigenschaft `.value`.
