# Aufgabe 1

Erstelle eine neue Seite (Page), die über einen Link im Menü erreichbar ist. Auf dieser Seite soll das aktuelle Datum angezeigt werden, welches sich in einer Property des zugehörigen PageModel befinden soll.

# Aufgabe 2

Erstellt auf der Seite ein Formular (`<form>`) mit einem Textfeld, wo der Benutzer seinen Namen eintragen kann, und einem Button.
Wenn der Benutzer diesen Button drückt, dann soll ein `GET`-Request zum Webserver geschickt werden. Dabei wird der Inhalt des Textfelds im Query-String übertragen (z.B. `name=Petra`). Der Webserver soll dann eine Page ausliefern, auf der der Benutzer mit Namen begrüßt wird.
